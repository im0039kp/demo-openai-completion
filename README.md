# demo-openai-completion

## 実行環境変数

- python .python-version を参照
- node .node-version .nvmrc を参照

## 準備

- python

  ```
  python -m pip install -r requirements.txt
  ```

- typescript
  ```
  npm install
  ```

## .env ファイル定義

.env ファイルを作成して

```
API_KEY_OPENAI="OpenAIのAPIキーを定義"
```

## 実行方法

- python
  VSCode から直接ファイルを実行

- node
  npx ts-node chat.ts を実行
